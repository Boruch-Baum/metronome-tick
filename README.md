# tick
A preset-oriented metronome TUI.

## Installation
Build time dependencies:
- [alsa-lib](https://www.alsa-project.org/wiki/Download)
- [scdoc](https://sr.ht/~sircmpwn/scdoc/)
- [GNU Make](https://www.gnu.org/software/make/#download)
- A C compiler

```
make
doas make install
```

To uninstall
```
doas make uninstall
```
